### Project 2 - Exploratory Data Analysis (EDA)
- Performed EDA on student performance dataset with 1000 records
- Analyzed gender wise and parental education wise performance
- Created histograms, box plots, scatter plots and heatmaps
- Found strong correlation between reading and writing scores
- Males perform better in math, females excel in reading and writing

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
